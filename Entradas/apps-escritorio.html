Conocimientos para crear programas en Windows

Si estamos pensando en empezar a programar y crear programas desde Windows, esto es todo lo que necesitaremos para poder lograr nuestro objetivo.

No hay un lenguaje de programación mágico para crear programas para Windows. Desde el sistema operativo de Microsoft podemos ejecutar prácticamente cualquier tipo de lenguaje de programación, siempre y cuando tengamos el compilador y el intérprete correspondiente.

Sin embargo, sí que hay varios lenguajes de programación que predominan en este sistema operativo, y son generalmente los de Microsoft. Por ejemplo, la mayoría de los programas con entorno gráfico desarrollados para Windows utilizan los lenguajes Visual, como Visual C++, Visual C#, Visual J#, y Visual Basic .NET.

Si vamos a crear un programa más sencillo que no necesita interfaz, C es una gran opción. Incluso C++ si queremos usar alguna de las librerías gráficas que hay disponibles para este lenguaje. La diferencia entre C y C++ es que el primero no está orientado a objetos, además de ser bastante más limitado que C++. Hoy en día hay grandes proyectos escritos en este segundo lenguaje (Chrome, Firefox, muchos juegos, etc), por lo que, sin meternos de lleno en lenguajes Visual, es muy cómodo programar en él.

Si queremos crear un programa para Windows, pero que también funcione en otros sistemas operativos, entonces debemos usar otros lenguajes de programación multiplataforma. Java, por ejemplo, es uno de los más recurridos ya que todo lo que hagamos en este lenguaje podrá funcionar sin problemas en cualquier sistema operativo. Python, por otro lado, es un lenguaje similar muy popular últimamente para crear programas multi-plataforma.

También es fácil encontrar desarrollos en otros lenguajes, como Object Pascal, Ruby u Objective-C.

Dependencias del lenguaje

Dependiendo del tipo de programa que vayamos a crear, podemos instalar unas u otras dependencias para poder trabajar sin problemas.

  • NodeJS.- En caso de usar este framework para el desarrollo del software, tendremos que instalarlo tanto en Windows como en el Subsistema de Windows para Linux.
  • Python.- Igual que NodeJS, también tendremos que instalar todas las librerías de Python en Windows o en WSL para poder desarrollar y ejecutar estos programas.
  • Android.- Podemos instalar Android Studio para crear apps para Android, o instalar otras opciones multiplataforma, como Xamarin, React o Cordova, para la creación de los programas.

Para el resto de lenguajes, como C, C++, C#, Java y toda la plataforma Windows UWP, bastará con instalar los IDE de programación de Visual Studio, ya que estos traerán todo lo necesario para dar forma a nuestros programas, diseñar las interfaces y mucho más.

Kit esencial de cualquier desarrollador de Windows

En realidad, podríamos programar sin necesidad de necesitar ningún software especial, tan solo con un editor de texto como puede ser el Bloc de Notas. Sin embargo, hoy en día existen programas (IDE) muy completos que, además de ayudarnos a programar, nos permiten compilar el código para convertir nuestras líneas en algo funcional.

Visual Studio Code

Visual Studio Code es un editor de texto que cuenta con muchas funciones de programación. Este programa cuenta con una gran cantidad de plugins que nos permiten adaptarlo para prácticamente cualquier lenguaje de programación que queramos. Gracias a sus funciones de IntelliSense, este programa puede resaltar la sintaxis, ayudarnos a auto-completar código e incluso encontrar errores.

Se integra con Git, es totalmente personalizable y, además, de código abierto. Podemos descargar Visual Studio Code para Windows, Linux y macOS.

Microsoft Visual Studio

Visual Studio es un IDE mucho más avanzado que Visual Studio Code desarrollado por Microsoft. Este IDE está desarrollado especialmente para la creación de programas para Windows, y es compatible con los principales lenguajes Visual, como ASP.NET, Visual C++, Visual C#, Visual J#, y Visual Basic .NET.

Este IDE nos permite crear fácilmente aplicaciones de servidor y software de escritorio para todas las versiones de Windows. Con los conocimientos adecuados es posible incluso crear aplicaciones (y juegos) 3D con él.

Windows Template Studio

Windows Template Studio es una extensión para Visual Studio diseñada especialmente para facilitar la creación de aplicaciones universales (UWP) para Windows 10. Este plugin se integra dentro del IDE de Microsoft y, con unos clics, nos permite dar forma al programa que queramos.

Azure y .NET

Aunque ya con esto entramos en temas un poco más avanzados, cualquier desarrollador que tenga en mente un proyecto a gran escala debe contar, lo primero, con la nube. Y qué mejor nube para desarrollar dentro de Microsoft que la suya: Azure.

Esta nube nos ofrece todo lo necesario para poder dar forma a cualquier proyecto, además de una serie de funciones y herramientas para integrar nuestras plataformas en ellas, llevar a cabo pruebas, continuar los desarrollos y administrar todas nuestras apps.

Y, junto a Azure, no podemos olvidarnos de .NET Core. Esta plataforma de código abierto cuenta con todas las herramientas y bibliotecas necesarias para crear cualquier tipo de app, tanto para PC como para móvil o que se ejecute en web. Además, también se utiliza para creación de juegos, para el Internet de las cosas y para el despliegue de microservicios.

Alternativas

Otros software interesante para crear programas desde Windows:

NetBeans

NetBeans es otro IDE de programación, esta vez enfocado principalmente al desarrollo de programas en Java. Este IDE es de código abierto y se caracteriza principalmente por ser modular, ya que cada usuario puede adaptarlo a sus necesidades mediante «módulos».

Necesitamos tener instalado el Java SDK para poder programar y compilar código escrito en este lenguaje. La ventaja es que todo lo que programemos en él funcionará igualmente en otros sistemas operativos.

Además de programar en java, con los módulos adecuados podemos programar también en JavaScript, HTML5, PHP o C/C++, entre otros.

Delphi

Otro IDE de programación muy popular entre los programadores es Delphi. Este IDE nos ayuda a crear aplicaciones multiplataforma compiladas de forma nativa (sin tener que recurrir a otros programas o compiladores adicionales) de manera que el resultado final sea lo más puro posible.

Este IDE se puede utilizar para programar en varios lenguajes de programación, aunque lo más común es usarlo para la programación en Object Pascal. Todo lo que escribamos y compilemos con él será compatible con Windows y con Linux. También es muy utilizado para el desarrollo web, permitiendo crear bases de datos cliente-servidor y aplicaciones y servicios web, como CGI, ISAPI, NSAPI, COM y DCOM.

Notepad++

Por último, no podíamos olvidarnos de Notepad++. Este editor de texto gratuito y de código abierto es una de las mejores alternativas al Bloc de Notas de Windows, pero también cuenta con complementos que nos permiten convertirlo en un completo IDE de programación. Si queremos algo sencillo de utilizar para programar a la antigua usanza (en modo texto, sin IDE), este es nuestro editor.

Windows Terminal

Windows nos ofrece como apps diferentes CMD y PowerShell. Sin embargo, trabajar con ellas no es precisamente fácil e intuitivo, especialmente si necesitamos varias instancias de cada una de ellas. Por ello, Microsoft ha desarrollado Windows Terminal, una app gratis y de código abierto desde la que vamos a poder ejecutar CMD, PowerShell, Bash y Azure desde una sola ventana, todo separado en pestañas.

Microsoft PowerToys

Las nuevas Microsoft PowerToys son un conjunto de programas y herramientas resucitadas de la era de Windows 95 gracias a las cuales vamos a poder llevar nuestra productividad a un nuevo nivel. Podemos crear escritorios virtuales, mapear el teclado y mucho más para que la tarea de programación sea lo más cómoda e intuitiva posible.

WinGet

Por último, de sobra se sabe que Linux es un sistema operativo superior a la hora de desarrollar, compilar y montar infraestructuras de servicios. Aunque siempre podemos tener nuestro propio servidor independiente, si usamos Windows 10 como PC principal vamos a poder virtualizar nuestro propio Linux gracias al Subsistema de Windows para Linux, o WSL.

WSL

Instalar paquetes, programas y dependencias desde Linux es muy sencillo, pero en Windows la cosa se complica. Por ello, podemos recurrir a WinGet, o Windows Packet Manager, para poder instalar, actualizar y desinstalar todo tipo de programas desde CMD, igual que si lo hiciéramos con el comando APT de Linux.

Así vamos a poder usar herramientas y comandos de Linux desde CMD o PowerShell sin necesidad de depender de más equipos. Todo en un mismo lugar.

¿Qué sistema operativo necesitamos para programar?

Además del software, también es necesario contar con el sistema operativo adecuado. Lo ideal es que si vamos a crear un programa para Windows tengamos una versión actualizada de Windows 10, ya que así podremos tener el último SDK y nuestro programa, además de tener la mejor compatibilidad, estará a la última.

Lo mismo se aplica en el caso de crear un programa para Linux, o para macOS (aunque en este último caso no queda otra).

Además, en el caso de usar Windows 10, es recomendable tener la edición PRO, ya que ofrece una serie de herramientas avanzadas que nos pueden ser de mucha utilidad.

¿Puedo programar para Windows en otros sistemas operativos? Por supuesto. Sobre todo si programamos en texto plano y no dependemos de IDEs avanzados como Visual Studio. Pero, si queremos compilar y depurar el programa, lo ideal es contar con el sistema correcto.
